# You vs You

You vs You ist eine Web-App zur persönlichen Selbstverbesserung.  
Der Fokus liegt darauf, sich nicht mit anderen zu vergleichen, sondern kontinuierlich besser zu werden als das eigene frühere Ich.

---

## Ziel des Projekts

You vs You kombiniert spielerische Elemente mit praktischen Tracking-Tools, um Motivation, Disziplin und Selbstreflexion langfristig zu fördern.  
Alles ist in einer einzigen Anwendung gebündelt und an einen Nutzeraccount gekoppelt.

---

## Features

| Feature | Beschreibung |
|------|-------------|
| Minigames | Spiele zur Verbesserung von Reaktionszeit, Zielgenauigkeit und Gedächtnis |
| Kalorientracker | Tracking von Kalorien, Nährwerten und Makros |
| Habit Tracker | Erstellen und Verfolgen von Gewohnheiten |
| User-System | Login, Registrierung und persistente Datenspeicherung |
| Backend | Supabase für Auth, Datenbank und Serverfunktionen |

---

## Minigames

Die Minigames sind ein zentraler Bestandteil der App und dienen der kognitiven Leistungssteigerung.

| Spiel | Zweck |
|----|----|
| Reaction Time | Messung und Verbesserung der Reaktionszeit |
| Aim Trainer | Training von Zielgenauigkeit |
| Number Memory | Merken und Wiedergeben von Zahlenfolgen |
| Visual Memory | Visuelles Gedächtnistraining |
